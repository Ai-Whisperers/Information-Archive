# Dashboard and Integration Rules (41-50)

## Rule 41: Central Repo View
Central Ignition view lists all repos with task counts by status.

## Rule 42: Filters
Filters for role, priority, repo status (active/inactive).

## Rule 43: Priority Color Coding
Color-code priorities:
- **Red**: High
- **Yellow**: Medium
- **Green**: Low

## Rule 44: Drill-Down Subtasks
Drill-down expandable rows for subtasks.

## Rule 45: Role-Based Security
Role-based security: View-only for non-assignees.

## Rule 46: Git Auto-Refresh
Git bindings auto-refresh task data every 15 mins.
