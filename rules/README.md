## Quick Reference

### Priority Levels
- **High** (Red): Urgent or blocks others
- **Medium** (Yellow): Important
- **Low** (Green): Enhancements

### Status Options
`To Do` → `In Progress` → `Review` → `Done` | `Blocked` | `Outdated`
